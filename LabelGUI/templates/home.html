Welcome to DroneAI

This is a collaborative labeling platform for drone video datasets. It supports validation labeling, training data collection, crash verification, shared database workflows across computers, and real-time team communication through MQTT.

Validation Labeling

Label validation events such as crashes, takeoff, or landing from videos. You can start from Excel for team workflows or use a manual link for quick testing.

Training Data

Create labeled training data for machine learning by assigning labels to video segments and exporting structured training output.

Crash Analysis & Verification

Review predicted crash counts and compare them against human verification results. This supports validation of automated crash detection output.

Open Crash Analysis

Database & Team Collaboration

Export and import the shared database so team members on different computers can continue previous work. Configure MQTT to see labeling activity from teammates in real time.

DB Tools MQTT Live Team Feed

Team Workflow

Before starting new labeling work, import the latest shared database from DB Tools. Then complete validation, training, or crash verification tasks. After finishing, export the updated database so the next teammate can continue from the latest version. MQTT can be used to show who is currently working in real time.
